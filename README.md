Este é um projeto que utilizei para a matéria de Teste de Software da Faculdade. É um jogo simples de labirinto, feito em JAVA e GUI como interface gráfica,
ainda em processo de evolução.
Para executar basta iniciar a main e uma Interface Gráfica irá aparecer e o jogo incializará.

Comandos:
  Os comandos devem ter obrigatoriamente como primeiro argumento a palavra "go" seguido de uma direção da rosa dos ventos. 
  * north
  * south
  * west
  * east
  Haverá na interface um espaço para digitar os comandos.
Objetivo:
  Haverá paredes e limites no mapa, evite-os e chegue ao objetivo final que será representado por um bloco no mapa de cor única.
